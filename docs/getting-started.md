## Accessing the System

### Login

1. Navigate to the application URL
2. Enter your email address
3. Enter your password
4. Click **Sign In**

### Default Credentials

For first-time setup, use the admin credentials:

| Field | Value |
|-------|-------|
| Email | admin@tertiaryinfotech.com |
| Password | 123456 |

All test accounts use password `123456`. See the [README](../README.md) for the full list of test accounts.

{: .warning }
> Change the default password immediately after your first login for security.

## Navigation

### Sidebar Menu

The left sidebar provides access to all main features:

- **Dashboard** - Home page with overview metrics
- **Staff Directory** - Employee management
- **Leave** - Leave requests and balances
- **Expenses** - Expense claims
- **Payroll** - Salary and payslips
- **Calendar** - Events and schedules

### User Menu

Click your profile icon in the top-right corner to:

- View your profile
- Access settings
- Sign out

---

## First Steps

### 1. Update Your Profile

After logging in, update your profile information:

1. Click your profile icon
2. Select **Profile**
3. Update your details
4. Click **Save**

### 2. Explore the Dashboard

The dashboard shows:

- Total employees count
- Pending leave requests
- Pending expense claims
- Recent activity

### 3. Use the AI Assistant

Click the chat icon in the bottom-right corner to:

- Ask HR policy questions
- Get CPF calculation help
- Learn about leave entitlements
