## Viewing Employees

### List View
Displays employees in a table format with:
- Employee ID
- Name
- Department
- Position
- Status

### Grid View
Shows employees as cards with profile photos and basic info.

Toggle between views using the icons in the top-right corner.

## Managing Employees

### Add New Employee (HR/Admin)

1. Click **Add Employee**
2. Fill in personal details
3. Set employment information
4. Configure salary (optional)
5. Click **Save**

### Edit Employee

1. Navigate to employee profile
2. Click **Edit**
3. Update information
4. Click **Save Changes**

### Deactivate Employee

1. Navigate to employee profile
2. Click **Actions** > **Deactivate**
3. Confirm the action
4. Set end date

{: .warning }
> Deactivating an employee preserves their records for historical purposes.
