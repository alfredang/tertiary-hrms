## Leave Types

| Type | Default Days | Description |
|------|--------------|-------------|
| Annual Leave | 14 | Paid vacation leave |
| Sick Leave | 14 | Illness without medical certificate |
| Medical Leave | 60 | Extended leave with MC |
| Compassionate Leave | 3 | Bereavement leave |

---

## Leave Balances

### Viewing Balances
Navigate to **Leave** > **My Balance** to see:
- Entitlement for each leave type
- Days taken
- Days remaining

### Balance Calculation
```
Remaining = Entitlement - Taken - Pending
```

Balances reset annually on January 1st.

---

## Applying for Leave

### Step-by-Step

1. Go to **Leave** > **Apply**
2. Select **Leave Type**
3. Choose **Start Date** and **End Date**
4. The system calculates working days automatically
5. Enter **Reason** (required)
6. Click **Submit**

### Leave Request Statuses

| Status | Description |
|--------|-------------|
| Pending | Awaiting approval |
| Approved | Leave confirmed |
| Rejected | Leave denied |
| Cancelled | Withdrawn by employee |

{: .note }
> Half-day leave can be selected for partial days.

---

## Approving Leave (Managers)

### Approval Process

1. Navigate to **Leave** > **Pending Approvals**
2. Review the request details
3. Check team calendar for conflicts
4. Click **Approve** or **Reject**
5. Add comments (optional)

### Approval Notifications
- Email sent to employee on approval/rejection
- Calendar event created automatically on approval
- Leave balance updated immediately

## Leave Policies

### Advance Notice
- Annual Leave: 7 days advance notice
- Sick Leave: Same day notification acceptable
- Medical Leave: MC required for 2+ consecutive days

### Carryover
- Maximum 5 days can be carried to next year
- Must be used by March 31st

### Encashment
Annual leave encashment available upon:
- Year-end (limited days)
- Employment termination
