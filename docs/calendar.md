## Calendar Views

### Month View
Default view showing the entire month with events displayed on their respective dates.

### Week View
Detailed view of a single week with time slots.

### Day View
Hour-by-hour schedule for a specific day.

### Agenda View
List of upcoming events sorted by date.

| Type | Color | Description |
|------|-------|-------------|
| Holiday | Red | Public holidays, company holidays |

## Viewing Events

### Click an Event
Shows event details:
- Title
- Date and time
- Duration
- Description
- Participants (if applicable)

### Filtering
Filter events by:
- Event type
- Department
- My events only

## Creating Events (HR/Admin)

### Add New Event

1. Click on a date or **+ Add Event**
2. Enter **Title**
3. Select **Event Type**
4. Set **Start Date/Time**
5. Set **End Date/Time**
6. Toggle **All Day** if applicable
7. Add **Description** (optional)
8. Select **Color** (auto-assigned by type)
9. Click **Save**

### Recurring Events

Set up recurring events:
- Daily
- Weekly
- Monthly
- Yearly
- Custom pattern

## Calendar Integration

### Export to External Calendar

1. Click **Export**
2. Choose format:
   - ICS (iCal)
   - Google Calendar
   - Outlook
3. Download or sync

### Subscribe to Calendar
Get a subscription URL to sync with your personal calendar app.
